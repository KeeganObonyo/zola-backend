#The following are the requirements for the project:
    -Java --version 1.8.0
    -Scala --version 
    -sbt --version 1.9.0

Follow these steps to get started:

1. Launch SBT:

        $ sbt

2. Compile everything and run all tests:

        > test

3. Start the application:

        > start

4. Test the API at http://localhost:8080
